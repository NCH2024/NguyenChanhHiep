#Kỹ năng

  1. Ngôn ngữ lập trình C

    Lập trình cơ bản, hiểu về cú pháp, cấu trúc điều khiển, và xử lý chuỗi.

  2. Ngôn ngữ lập trình C++

    Nắm bắt các khái niệm hướng đối tượng như class, object, kế thừa và đa hình.

  3. CSS (Cascading Style Sheets)

    Thiết kế và định dạng giao diện trang web: bố cục, màu sắc, hiệu ứng cơ bản.

  4. Phát triển web với ASP.NET (Visual Studio 2013)

    Xây dựng và triển khai website động bằng công nghệ ASP.NET, kết hợp cơ sở dữ liệu SQL.

  5. Quản lý mã nguồn với GitHub

    Đẩy, kéo mã nguồn và quản lý dự án nhóm qua Git/GitHub cơ bản.

  6. Ngôn ngữ lập trình JavaScript

    Tạo các hiệu ứng động và tương tác trên website, xử lý DOM.

  7. Quản lý cơ sở dữ liệu với Microsoft SQL Server

    Thao tác với cơ sở dữ liệu: viết câu lệnh SQL, truy vấn, thêm/xóa/sửa dữ liệu.

  8. .NET Framework

    Nắm vững cách xây dựng ứng dụng web và sử dụng các thư viện có sẵn trong .NET.

  9. Sử dụng Visual Studio Code

    Sử dụng IDE để viết code nhanh chóng, tích hợp tiện ích hỗ trợ lập trình.
